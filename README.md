# React To-Do App

A modern, interactive To-Do application built with **React**, featuring drag-and-drop task reordering, dark mode, editable tasks, persistent storage with LocalStorage, and a fun confetti effect when completing tasks.

## Live Demo

[https://ArkapravaChanda.github.io/react-todo-app](https://ArkapravaChanda.github.io/react-todo-app)

---

## Features

- **Add, Edit, Delete Tasks:** Manage your tasks easily.
- **Drag-and-Drop Reordering:** Rearrange your active tasks effortlessly.
- **Completed Tasks at Bottom:** Finished tasks automatically move to the bottom.
- **Dark / Light Mode Toggle:** Switch between themes for day or night usage.
- **Persistent Storage:** Your tasks and theme settings are saved in LocalStorage.
- **Confetti Animation:** Celebrate task completion with a fun visual effect.
- **Responsive Design:** Works well on both desktop and mobile screens.
- **Hover Animations:** Subtle hover effects on completed tasks for a polished look.
